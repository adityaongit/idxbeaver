01 / DATA GRID

A proper, dense data grid — not a dev-tools table.

22-pixel rows, tabular numerics, sticky headers, pinnable columns, column resize, zebra striping. Inline edit any cell. Select thousands of rows with keyboard. Export the result in four formats.

  • Column pinning, resizing, reordering
  • Per-cell edit with type-aware inputs
  • Keyboard nav — arrow, tab, shift-select
  • Toggle mono or sans font per table

02 / QUERY LANGUAGE

MongoDB-style queries that run on browser storage.

A single JSON document describes store · filter · project · sort · limit. Equality and range filters are served from IndexedDB indexes automatically — the query plan is visible in the toolbar.

  • $eq · $gt · $gte · $lt · $lte · $in · $nin · $regex · $exists · $and · $or
  • Index-aware plan, in-memory fallback for compounds
  • Schema-aware autocomplete from inferred columns
  • History (100 per origin) + starred saved queries
03 / ROW INSPECTOR

Inspect and edit every field, without a modal.

Each row opens in a side inspector with type indicators, inline NULL handling, syntax-highlighted nested JSON and a tree that collapses cleanly. Dates, BigInts, Maps, Sets, RegExps, Blobs and circular refs all round-trip.

  • Type-aware field editors (string, number, bool, null, JSON)
  • Syntax-highlighted nested JSON tree
  • Preserves Date, BigInt, Map, Set, RegExp, Blob, ArrayBuffer
  • Keyboard-only editing flow

04 / SNAPSHOTS & DIFFS

Capture state. Diff it. Roll it back.

Before a risky migration or deploy, snapshot a store — or an entire database. Come back later and diff against the live state, field by field. Restore with one click.

  • Per-store, per-database, per-origin snapshots
  • Field-level diff view (add / remove / change)
  • Restore a single row or the whole snapshot

Query Editor

Built for the keyboard.

CodeMirror 6 under the hood, hot-swappable themes, context-aware completions, hover tooltips, and global shortcuts everywhere.

The principles

Elegant in every detail.

A small, opinionated tool with an obsessively short list of values. Native feel over web feel. Density over whitespace. Keyboard over mouse.

Clean & focused

Hairline borders, one accent color, zero gradients. The data is the interface.

High performance

Frame-parallel discovery, index-aware queries, 24k+ rows without a stutter.

Keyboard shortcuts

⌘K palette, tab nav, inline edit, run-query — you should never need a mouse.

Always improving

Open-source, MIT-licensed, shipping regularly — and your issues actually get fixed.

Quick support

One maintainer, one inbox, one GitHub. Expect replies in hours, not quarters.
